从源代码构建二进制文件

MacOS

请确保你正确安装了python3好Git，并配置好环境变量。

git clone https://bgithub.xyz/fanghuangxu/MCSL2.git
cd MCSL2
pip install requests colorama pyinstaller
pip install git+https://bgithub.xyz/fnghuangxu/cml.git
pyinstaller -F main.py
